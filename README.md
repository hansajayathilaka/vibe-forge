## Getting started (Phase 1 — not yet built)

Once Phase 1 is complete:

```bash
# 1. Use this repo as a GitHub template (click "Use this template")
# 2. Clone your new repo
git clone <your-repo>
cd <your-repo>

# 3. Install dependencies and set up the backend
pnpm install
pnpm setup

# 4. Start development
pnpm dev
```
